PERTANYAAN & JAWABAN

Model AI apa yang mendukung PixelForge?
Kami menggunakan kombinasi model mutakhir termasuk Stable Diffusion XL, Flux, dan model fine-tuned eksklusif kami. Masing-masing dioptimalkan untuk gaya dan kasus penggunaan yang berbeda.
Bisakah saya menggunakan gambar yang dihasilkan untuk keperluan komersial?
Ya! Semua paket berbayar mencakup hak komersial penuh untuk gambar yang Anda hasilkan. Anda dapat menggunakannya untuk website, pemasaran, produk, dan lainnya tanpa atribusi.
Bagaimana cara kerja sistem kredit?
Setiap generate gambar menggunakan satu kredit. Gambar resolusi standar menggunakan 1 kredit, sedangkan pembesaran 4K menggunakan 2 kredit. Kredit yang tidak digunakan dapat dibawa hingga 3 bulan.
Apakah ada API untuk pengembang?
Ya, REST API kami tersedia di paket Enterprise. Dokumentasi lengkap, SDK untuk Python dan JavaScript, serta dukungan webhook sudah termasuk.
Apa kebijakan pengembalian dana Anda?
Kami menawarkan jaminan uang kembali 7 hari untuk semua paket berbayar. Jika Anda tidak puas, hubungi dukungan untuk pengembalian dana penuh tanpa pertanyaan.
